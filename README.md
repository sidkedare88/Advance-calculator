# Advance Calculator

A modern calculator built using **HTML, CSS, and JavaScript**.

## Features

* Basic arithmetic operations
* Percentage calculation
* Responsive design
* Clean and modern interface

## Technologies

* HTML5
* CSS3
* JavaScript

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/sidkedare88/Advance-calculator.git
   ```
2. Open `calculator.html` in your browser.
